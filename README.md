## Getting Started
- Clone this repo
- Download the [SDL2 development library](https://github.com/libsdl-org/SDL/releases) and follow [this documentation](https://crates.io/crates/sdl2) to put the binary and header files in the right locations
- Run this command
```
cargo run
```
